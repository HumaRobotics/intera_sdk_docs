__init__(self, joint_angles=[], active_endpoint=None, options=None)
(Constructor)

source code 

Create a MotionWaypoint object. All parameters are optional.

Parameters:
  • joint_angles - the joint angles to store in the waypoint. If set to empty, then create an empty waypoint. If set to None, then use the default joint angles.
  • active_endpoint - the pose is computed using the active endpoint If set to None, then use default active endpoint
  • options - waypoint options. If set to None, then use default waypoint options
Returns:
a intera Waypoint.msg object with default values

set_joint_angles(self, joint_angles=[], active_endpoint=None, perform_fk=False)

source code 

All parameters are optional. If ommitted or set to None, use default.

Parameters:
  • joint_angles - the joint angles to store in the waypoint. If set to empty, then create an empty waypoint.
  • active_endpoint - the pose is computed using the active endpoint
  • perform_fk - boolean to determine if FK should be performed with the input joint angles

set_cartesian_pose(self, pose=None, active_endpoint=None, joint_angles=[])

source code 

All parameters are optional. If ommitted or set to None, use default.

Parameters:
  • pose - Cartesian position and orientation (PoseStamped msg) If set to empty, then create an empty waypoint.
  • active_endpoint - the pose is computed using the active endpoint
  • joint_angles - the joint angles to store in the waypoint. This is used as the nullspace bias if set.
